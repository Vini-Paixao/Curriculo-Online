Minhas Experiências

Website responsivo WordPress

Posso criar um website institucional, um website de comércio eletrônico e uma landing page para você usando a plataforma WordPress. Integrado com os melhores plugins do mercado, fazendo um site totalmente responsivo, rápido e com excelente SEO.

Codificação em diferentes linguagens

Com minhas habilidades em programação, posso ajudar a codificar programas em C#, com um banco de dados eficiente usando MySQL. Além disso, posso ajudar com a programação web, utilizando linguagens como PHP, HTML5, CSS3, JavaScript e ASP.NET, para criar uma experiência online rica e envolvente.

Textos para Mídias Sociais

Posso ajudar você com a elaboração de criativos e copy para campanhas de e-mail marketing, tanto para tráfego pago quanto orgânico. Também posso ajudar na segmentação do público-alvo e na criação de campanhas online eficazes utilizando as plataformas atuais de geração de texto.

Artes para Mídias Sociais

Desenvolvimento de materiais gráficos, redação publicitária e elaboração de conceitos, implementação e administração de publicações em diferentes plataformas de mídia social.
